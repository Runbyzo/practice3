Описание языка ассемблера для варианта 13
=

Разработанный язык ассемблера предназначен для описания программ для упрощённой учебной вычислительной машины (УВМ) в рамках варианта 13. Язык является человеко-читаемым представлением машинных команд и используется в качестве входного формата для ассемблера.

Программа представляет собой текстовый файл, состоящий из последовательности строк. Каждая строка может содержать:

- одну инструкцию ассемблера;

- комментарий;

- либо быть пустой.

Каждая инструкция занимает ровно одну строку.

Общий формат инструкции:
-
><мнемоника> <аргумент> [; комментарий]

Поддерживаемые команды
-
| Мнемоника | Назначение                            | Поле A | Размер (байт) | Аргумент   |
| --------- | ------------------------------------- | ------ | ------------- | ---------- |
| `CONST`   | Загрузка константы в аккумулятор      | 145    | 3             | константа  |
| `LOAD`    | Загрузка значения из памяти           | 193    | 4             | адрес      |
| `STORE`   | Запись аккумулятора в память          | 201    | 4             | адрес      |
| `NEG`     | Арифметическое отрицание аккумулятора | 161    | 2             | формальный |

Все команды в языке имеют один аргумент. Для унарной команды NEG аргумент является формальным и должен быть указан (обычно 0).

Числовые литералы
-
Аргументы команд могут задаваться в различных системах счисления:

- десятичной (10);

- шестнадцатеричной (0x10);

- восьмеричной (0o10).
